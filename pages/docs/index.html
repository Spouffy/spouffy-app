Subrary documentations

  • FUNCTION : getElm()

    returns html element (target can be a class name or id or tag name)

    getElm('body')

  • FUNCTION : hide(target)

    set target's display to none (uses getElm() targeting system)

    hide('hideElm')
    element

  • FUNCTION : show(target)

    set target's display to block (uses getElm() targeting system)

    show('showelm')
    element

  • FUNCTION : ElmFollowCursor(delay,id)

    set target's display to block (uses getElm() targeting system)

  • FUNCTION : pgcd(integer, integer)

    returns the commun devider of two numbers

  • FUNCTION : randomInt(minimum, maximum)

    returns a random integer between min and max parameters (DO NOT USE FOR SECURITY)

  • FUNCTION : createElm(tagName, parentTarget)

    create and returns an html element by giving the tag name and the parent (getElm() targeting system)

  • FUNCTION : on(event, target, function)

    puts Event Listener on any element (getElm() targeting system) and passes an anonym function

  • FUNCTION : load(target, file_path)

    dynamicly adds html page on html element (getElm() targeting system)
